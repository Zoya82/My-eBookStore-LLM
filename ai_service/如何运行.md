# ai_service 运行手册（柴子萱）

环境已经装好（`code/My_eBookStore/.venv`），以后按下面用即可。

## 每次启动服务（3 步）

在「终端」里：

```bash
cd "/Users/yunye/Documents/课程/大三下/软件工程引论/课程设计/code/My_eBookStore"
source .venv/bin/activate
uvicorn ai_service.main:app --reload --port 8001
```

然后浏览器打开 **http://localhost:8001/docs** ，点接口 →「Try it out」→「Execute」测试。
关闭服务：终端里按 `Control + C`。

## 换 API Key

编辑 `ai_service/.env`，改 `DASHSCOPE_API_KEY=` 后面的值，保存即可（不用重启会自动生效；没生效就重启服务）。

```bash
open -e ai_service/.env
```

## 只想快速自检（不启动服务）

```bash
python -m ai_service.test_llm
```

看到 `✅ 大模型调通！` 即正常。

- **提示词**：`ai_service/prompts.py`
- **样例图书**：`ai_service/data/sample_books.json`
- **接口字段**：`ai_service/schemas.py`
